Symbiosis

Category

VR Application

Project Date

Sept 2018 - Present

Skills

Unity, C#

Medium Article

Symbiosis is a project created by the Biofeedback team in Extended Reality at Berkeley. Our goal is to create a VR environment that uses biofeedback to assist individuals with anxiety. Our project has been accepted into both Oculus Start and Oculus Launchpad!

Created an interactive virtual reality application to aid individuals with anxiety through responsive biofeedback

Developed the virtual reality environment using Unity and Oculus VR tools

Utilized EEG and breathing data to create a responsive environment
